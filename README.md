# Sound! Euphonium

素晴らしいラジオ番組を録音するためのスクリプト。


## HOW TO USE
* "onsen_dl.py" を定期的に実行してください。
* 録画する番組を変更したいときは "onsen_dl.py" の
  Consts.TARGET_CHANNELS を適宜変更してください。
